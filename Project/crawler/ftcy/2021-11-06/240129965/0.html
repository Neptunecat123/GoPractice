写几点渣浪反黑净化注意事项

来自: 经年不见 2021-07-29 14:03:03

（我知道这玩意应该在微博数据站/反黑站进行科普，但是唯饭站我也不去，cp站不清楚我们这边怎么界定，大家随缘看吧）

1⃣️出现广场黑帖后应该最先做什么

1.1 点击右上角投诉。

勾选“有害信息-其他有害”，勾选“有害信息-暴力血腥”，如果是红v黄v，勾选“垃圾营销-昵称简介广告”（gct掉了快），勾选”人身攻击-攻击我”内容见模版：

「此微博用户利用艺人大名进行恶意辱骂、用词相当恶劣，影响了社区网络环境以及本人微博使用观感，违反了《微博社区管理规定》，请贵方严肃处理，谢谢！」

1.2 点击右上角，“分享“-”更多“-搜唯粉家的管理组。

留言让他们做反黑🔗（如果你会做也可以自己做好发，教学略复杂就不说了，如果真的想学我再教）

⚠️千万不要直接给【黑博】转发/回复（特别是辱骂），会帮助黑博反复上gc，上综合，甚至上热门😨

2⃣️第二步做什么

2.1 有gct的，优先发【原创微博】

#大名#🌿/☁️#各家自己的tag#

你在搜索 大名|缩写|昵称|惯称 ？

那一定不能错过（🌈内容）

@大名

ps. 如何测试gct？

发送：

#实时号测试# test 人名。

然后到实时去查，如果看的到，就是有。

（现在流传文案中带：“ipad 养乐多 旺仔” 可以帮助你上gc，可以414有些人确实可以）

ps.为什么带双tag？

一来双tag有效，二来防止对方脏词条，即输入完 “大名”后下方自动关联的内容需要进行维护。

2.2有无gct的，都转发评论【投票】博或目前单人的【热门】博

⚠️1.转发过程可以带#大名#，依旧算数据。

⚠️2.转发【投票】博，一定要删除“ id ”及“ 投票” 字样【分享】。

分享完成后取消投票可反复转发分享。（因为带这两样会降低你上gc的几率，同时也会影响原博权重以及原博上热门）

3⃣️第三步做什么

等待双方反黑博出来后，挨个【投诉】并【回复】。

ps1.为什么投诉完要回复？

因为可以帮助把这个博顶起来，让更多人看到，同时也为了让别人看到大家都在努力鼓舞士气。

⚠️1.回复时候，不要at正主（你不会想正主来看这些东西吧）

⚠️2.回复时候，不要留言“卡”（诸如：“卡”、“票”、“小号”等，都是敏感词，会被吞评，会影响博主权重，可回复词条、大名等进行顶帖打卡工作。）

（我只是做个科普而已，如果觉得在被教育dbq了，统一回复大家不必客气，能有点用就行）
